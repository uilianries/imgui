# Text
## ImGui::Text

Draws formatted text. The format options are like `printf`.

### Example
```c++
ImGui::Text("Hello world %f", 42);
```

# Main
## ImGui::Button
Draws a button and returns whether it was clicked or not.
### Example
```c++
if (ImGui::Button("Button"))
{
    printf("Button Clicked");
}
```
